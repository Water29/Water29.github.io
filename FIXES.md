## 💡 购买按钮说明

### 主页产品卡片结构

```
┌─────────────────────────┐
│       🍎 (图片)          │
│  时令水果                │
│  有机红富士苹果          │
│  绿源农场                │
│  2.5 FRT    150 件库存   │
│ ┌─────────┬───────────┐ │
│ │🛒 加入  │ ⚡ 立即购买│ │
│ │ 购物车  │           │ │
│ └─────────┴───────────┘ │
└─────────────────────────┘
```

### 两个按钮的区别

| 按钮 | 功能 | 跳转 |
|------|------|------|
| 🛒 加入购物车 | 添加到购物车 | 留在当前页 |
| ⚡ 立即购买 | 添加到购物车 + 结算 | 跳转到 checkout.html |

## 🔗 MetaMask 连接流程

```
用户点击"连接 MetaMask"
        ↓
检查是否安装 MetaMask
        ↓
    ┌───────┐
    │ 未安装 │──→ 显示安装提示
    └───────┘
        ↓ 已安装
请求账户访问权限
        ↓
用户在 MetaMask 确认
        ↓
显示钱包地址（脱敏）
        ↓
启用"确认支付"按钮
```

## ⚠️ 注意事项

### MetaMask 测试提示

1. **使用测试网** - 建议先用 Base Sepolia 等测试网
2. **获取测试币** - https://faucet.base.org
3. **不要泄露私钥** - 永远不要分享私钥或助记词

### 演示模式

如果合约未部署，支付会进入演示模式：
- 模拟交易处理（2 秒延迟）
- 生成随机交易哈希
- 清空购物车
- 显示成功页面
