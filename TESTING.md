## 完整购物流程测试

### 1. 访问首页
```
http://localhost:9000
```

### 2. 浏览产品
- 点击导航"🍇 全部产品"
- 或直接滚动到产品区域

### 3. 添加商品到购物车
```
商品 1: 有机红富士苹果 × 2
商品 2: 进口香蕉 × 1
```

### 4. 查看购物车
- 点击右上角购物车图标 🛒
- 确认商品和数量正确
- 确认费用明细：
  - 小计：6.8 FRT (2.5×2 + 1.8×1)
  - 燃油费：0.68 FRT
  - 总计：7.48 FRT

### 5. 结算
- 点击"💳 立即结算"
- 跳转到 checkout.html

### 6. 连接钱包
- 点击"🦊 连接 MetaMask 钱包"
- 在 MetaMask 弹窗中确认
- 确认显示钱包地址（如 `0x1234...5678`）

### 7. 支付
- 点击"💰 确认支付"
- 在确认对话框中查看：
  ```
  确认支付 7.48 FRT
  
  商品小计：6.80 FRT
  燃油费 (10%): 0.68 FRT
  ```
- 点击"确认"

### 8. 支付成功
- 查看成功弹窗
- 显示交易哈希：`0x...`
- 点击"🔗 查看区块链交易"

### 9. 验证库存扣减
- 点击"🛒 继续购物"
- 查看产品列表
- 确认库存变化：
  - 有机红富士苹果：150 → 148
  - 进口香蕉：200 → 199

### 10. 验证持久化
- 刷新页面
- 再次查看库存
- 确认库存数据保持

## 常见问题排查

### Q1: 费用显示 0 FRT
```javascript
// 在控制台运行
console.log(cart);
// 如果 cart 是空数组，重新添加商品
```

### Q2: 库存扣减后刷新恢复
```javascript
// 检查 localStorage
console.log(localStorage.getItem('fruitStoreProducts'));
// 如果是 null，说明保存失败
```

### Q3: 仍提示未安装 MetaMask
```javascript
// 检查 ethereum 对象
console.log(typeof window.ethereum);
// 应该返回 "object"
// 如果返回 "undefined"，检查扩展是否启用
```
